# TYPOGRAPHY_SPACING

## 文件用途
定義文字階層、間距與可讀性原則。

## 初版規格
- Display：32–40px，用於 Desktop 首頁 greeting。
- Title：24–28px，用於區塊標題。
- Body：16px，主要閱讀文字。
- Caption：12–14px，狀態與輔助資訊。
- Spacing：以 8px grid 為基礎，卡片內距 16–24px。

## UX 規則
- 今日摘要與提醒必須可快速掃讀。
- 不使用過細字重作為主要資訊。
- 夜間模式降低對比刺激，但不犧牲文字辨識。

## Responsive 規則
- Desktop 可使用較大的 greeting 與寬鬆間距。
- Tablet 保持 16px body。
- Mobile body 不低於 15–16px，區塊間距保留呼吸感。

## Stitch MCP 可理解描述
Use clean readable typography similar to modern iOS interfaces, generous spacing, calm hierarchy, no dense dashboard clutter.
