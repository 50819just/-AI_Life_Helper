# REMINDER_FLOW

## 文件用途
定義提醒功能從查看、建立到完成的流程。

## 初版規格
- 使用者查看 Today Reminders。
- 可透過手動輸入或 AI Assistant 建立提醒。
- 提醒包含標題、日期時間、重複規則、優先程度、備註。
- 完成後移入 Completed 或以淡化狀態保留。

## UX 規則
- 今日提醒需清楚分出緊急與普通。
- 建立提醒步驟要少，AI 可協助補齊時間語意。
- 錯過提醒需溫和提示，不製造壓力。

## Responsive 規則
- Desktop：清單與詳情可雙欄。
- Tablet：清單優先，詳情抽屜。
- Mobile：Today / Upcoming / Completed 分段或 tabs。

## Stitch MCP 可理解描述
Reminder screen with today reminders, upcoming list, completed state, gentle notification tone, AI-assisted reminder creation entry.
